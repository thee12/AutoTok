Local-first video workflow

InTheLoop helps operators create, review, and publish approved short-form videos.

InTheLoop is powered by AutoTok, a desktop and command-line workflow for turning approved source text and authorized media into reviewable vertical video packages.

Create Locally

Import approved story text, narration audio, subtitles, and authorized background media into a local workspace.

Review First

Preview the generated video package locally and approve it before any publishing command can run.

Publish Explicitly

Use official platform APIs only after human approval and an explicit publish confirmation.

TikTok Integration

InTheLoop uses Login Kit to authorize the operator's TikTok account and TikTok's Content Posting API to publish approved videos through Direct Post. The workflow is designed for the account owner to review and explicitly confirm publication.

  • Scopes used for TikTok: user.info.basic, video.upload, and video.publish.
  • Generated content is stored locally unless the operator chooses to publish.
  • The app does not automate comments, direct messages, follows, likes, or other engagement behavior.

Operator Responsibilities

Operators are responsible for using source text, audio, and media they own or are authorized to use. InTheLoop does not provide legal advice, guarantee reach, or determine whether specific content is eligible for monetization or platform distribution.
